
# 📘 Hangman Game Challenge

## 🎯 Objective

Build a classic Hangman game using Python to practice string manipulation, loops, conditionals, and user input handling.

## 📝 Tasks

### 🛠️ Create the Hangman game

#### Description

Write a Python program that lets a player guess letters in a hidden word. The program should select a word from a list, display progress using blanks and letters, and keep track of remaining attempts.

#### Requirements
Completed program should:

- Randomly choose a word from a predefined list
- Prompt the player to guess one letter at a time
- Display the current word progress using blanks and correctly guessed letters, for example:

```text
Word: _ A N _ M A N
``` 

- Track incorrect guesses and show the number of attempts remaining
- End the game with a win message when the word is fully guessed
- End the game with a lose message when attempts run out
- Prevent repeated guesses from counting as additional incorrect attempts

### 🛠️ Improve the player experience

#### Description

Add clear messages and friendly feedback to help the player follow the game. This includes showing guessed letters, remaining attempts, and game outcome.

#### Requirements
Completed program should:

- Show the letters already guessed by the player
- Display a message after each guess indicating whether it was correct or incorrect
- Update and display the number of remaining attempts
- Print a final message with the completed word when the game ends
